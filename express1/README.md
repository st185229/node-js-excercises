# readme
## setup
- create src folder and move js file to ts file and rename to .ts
- npm init -y
- npm install
- npm i --save-dev typescript
- npm i --save-dev ts-node
- npm i --save-dev @types/node
- npx tsc --init
- npm i --save-dev @types/express
- npm i --save-dev nodemon
## run
- npm run start
